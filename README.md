# 미니게임 플랫폼

Next.js 14 기반의 웹 미니게임 플랫폼입니다. 우리은행 WON PLAY를 벤치마크하여 개선한 사용자 참여형 게임 서비스로, 다양한 캐주얼 게임과 게이미피케이션 요소를 결합했습니다.

---

## 주요 특징

### 13개 미니게임
- **퍼즐**: 스도쿠, 2048, 메모리 카드, 슬라이딩 퍼즐
- **두뇌**: 지뢰찾기, 워들, 숫자야구
- **액션**: 플래피 버드, 뱀 게임, 벽돌깨기
- **기타**: 타이핑 테스트, 반응속도 테스트, 색상 맞추기

### 게이미피케이션 시스템
- **미션 시스템**: 일일/주간 미션 수행 및 포인트 보상
- **레벨 시스템**: 20단계 레벨 + 칭호 (뉴비 → 절대자)
- **업적 시스템**: 희귀도별 업적 (일반/레어/에픽/전설)
- **출석 체크**: 연속 출석 보너스 (3일/7일/14일/30일)

### 경쟁 요소
- 게임별 글로벌 랭킹 (Supabase 기반)
- 개인 통계 대시보드 (플레이 기록, 승률, 최애 게임 분석)

## 기술 스택

| 영역 | 기술 |
|------|------|
| 프레임워크 | Next.js 14 (App Router) |
| 언어 | TypeScript |
| 스타일링 | TailwindCSS |
| 백엔드 | Supabase (랭킹) |
| 상태 관리 | localStorage (미션, 포인트, 통계, 업적) |
| 테스트 | Jest + React Testing Library |
| 아이콘 | lucide-react |

## 빠른 시작

```bash
# 의존성 설치
npm install

# 개발 서버 실행
npm run dev

# 테스트 실행
npm test

# 프로덕션 빌드
npm run build
```

## 환경 변수

```env
# .env.local
NEXT_PUBLIC_SUPABASE_URL=https://your-project.supabase.co
```

## 프로젝트 구조

```
src/
├── app/           # 페이지 (Next.js App Router)
├── components/    # UI 컴포넌트
│   ├── games/     # 게임별 컴포넌트
│   ├── home/      # 메인 페이지
│   ├── mission/   # 미션 관련
│   └── ui/        # 공통 UI (Skeleton 등)
└── lib/           # 비즈니스 로직
    ├── games/     # 게임 로직
    ├── data/      # 정적 데이터
    └── *.ts       # 시스템 (미션, 업적, 레벨, 통계)
```
