# 릴리즈 노트

## v1.0.0 (2026-01-15)

미니게임 플랫폼 정식 출시

---

### 🎮 게임 라인업 (총 13종)

#### 퍼즐 게임
- **스도쿠** - 클래식 숫자 퍼즐 (난이도: 쉬움/보통/어려움)
- **2048** - 타일 합치기 무한 목표 시스템
- **메모리 카드** - 짝 맞추기 게임
- **지뢰찾기** - 클래식 지뢰찾기 (8x8, 10개 지뢰)
- **슬라이딩 퍼즐** - 15퍼즐 (3x3/4x4/5x5)

#### 두뇌 게임
- **워들** - 5글자 영단어 추측 게임
- **숫자야구** - 3자리 숫자 맞추기
- **타이핑 테스트** - 60초 타자 속도 측정

#### 액션 게임
- **반응속도 테스트** - 클릭/키보드 반응 측정
- **플래피 버드** - 장애물 피하기
- **뱀 게임** - 클래식 스네이크 게임
- **벽돌깨기** - 아케이드 브레이크아웃
- **색상 맞추기** - 제한 시간 내 색상 선택

---

### ✨ 주요 기능

#### 미션 시스템
- 일일 미션 (매일 자정 초기화)
- 주간 미션 (매주 월요일 초기화)
- 미션 완료 시 포인트 보상

#### 레벨 & 칭호 시스템
- 20단계 레벨 체계 (뉴비 → 절대자)
- 포인트 기반 레벨업
- 레벨별 고유 칭호 및 아이콘

#### 업적 시스템
- 50+ 업적 달성 가능
- 희귀도 등급: 일반 / 레어 / 에픽 / 전설
- 업적 해금 시 보상 포인트 지급

#### 출석 체크
- 매일 출석 보상
- 연속 출석 보너스 (3일/7일/14일/30일)
- 월별 출석 현황 캘린더

#### 랭킹 시스템
- 게임별 글로벌 랭킹
- 실시간 순위 업데이트
- Top 100 리더보드

#### 통계 대시보드
- 게임별 플레이 기록
- 총 플레이 시간, 승률 분석
- 최애 게임 자동 산출

---

### 🛠 기술 사항

- Next.js 14 (App Router)
- TypeScript + TailwindCSS
- Supabase (랭킹 데이터)
- localStorage (미션/통계/업적)
- 반응형 모바일 UI (max-w-lg)
