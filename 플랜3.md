# 추가 가능한 미니게임 플랜

## 1. 장애물 피하기 (Endless Runner)
**난이도: 쉬움 | 예상 구현 시간: 중간**

### 게임 설명
- 캐릭터가 자동으로 달리고, 장애물을 점프/슬라이드로 피하는 게임
- 시간이 지날수록 속도 증가
- 코인 수집으로 추가 점수

### 조작
- 스페이스/탭/클릭: 점프
- 아래 화살표/스와이프 다운: 슬라이드

### 점수 산정
- 생존 시간 × 10 + 코인 × 100

### 구현 요소
- Canvas 또는 CSS 애니메이션
- 충돌 감지
- 난이도 곡선 (속도 증가)

---

## 2. 플래피 버드 (Flappy Bird)
**난이도: 쉬움 | 예상 구현 시간: 짧음**

### 게임 설명
- 새가 파이프 사이를 통과
- 클릭/탭으로 위로 점프
- 파이프에 닿으면 게임 오버

### 조작
- 클릭/스페이스: 점프

### 점수 산정
- 통과한 파이프 수 × 100

### 구현 요소
- 중력 물리
- 파이프 생성 로직
- 충돌 감지

---

## 3. 스네이크 게임 (Snake)
**난이도: 쉬움 | 예상 구현 시간: 짧음**

### 게임 설명
- 뱀이 먹이를 먹으면 길어짐
- 벽이나 자기 몸에 닿으면 게임 오버
- 클래식 레트로 게임

### 조작
- 방향키/WASD: 방향 전환
- 스와이프: 모바일 조작

### 점수 산정
- 먹이 × 100 + 생존 시간 보너스

### 구현 요소
- 그리드 기반 이동
- 꼬리 성장 로직
- 충돌 감지

---

## 4. 브레이크아웃 (Breakout / 벽돌깨기)
**난이도: 중간 | 예상 구현 시간: 중간**

### 게임 설명
- 패들로 공을 튕겨 벽돌 파괴
- 모든 벽돌을 깨면 클리어
- 공을 놓치면 라이프 감소

### 조작
- 마우스/터치: 패들 이동
- 좌우 화살표: 키보드 조작

### 점수 산정
- 벽돌 × 10 + 클리어 보너스 + 남은 라이프 보너스

### 구현 요소
- 공 물리 (반사각)
- 벽돌 배치
- 아이템 (공 크기, 패들 크기, 멀티볼)

---

## 5. 색상 맞추기 (Color Match)
**난이도: 쉬움 | 예상 구현 시간: 짧음**

### 게임 설명
- 제시된 색상과 같은 색상 선택
- 텍스트와 색상이 다를 때 헷갈림 유발 (스트룹 효과)
- 제한 시간 내 최대한 많이 맞추기

### 조작
- 버튼 클릭/탭

### 점수 산정
- 정답 × 100 - 오답 × 50 + 시간 보너스

### 구현 요소
- 랜덤 색상 생성
- 스트룹 효과 적용
- 타이머

---
